<h1> Tiva-C Counter Croject </h1>

<ul>
  <li>
    This project is implemented using tm4c123gh6pm microcontroller, LCD, and switches.
  </li>
  <li>
    It's a simple counter that can be incremented, decremented, and reset using switches.
  </li>
</ul>
